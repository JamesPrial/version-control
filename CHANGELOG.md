## [1.0.0] - 2025-12-28

### Added
- git-ops agent (Haiku model) for lightweight Git operations

- github-actions-writer skill: workflow templates, validation, security audit
- 11 workflow templates (CI, CD, security, advanced)
- 6 reference guides (syntax, security best practices, troubleshooting)
